TLDR - nmap
===========

Overview
--------

[nmap] is a UNIX command that can explore the network and its security.

Examples
--------

- If you wanted to scan the notre dame website you would use:
	$ nmap -v nd.edu

- If you wanted to see the open ports on that domain you would use:
	$ nmap -v -open nd.edu
