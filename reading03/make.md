TLDR - make
=====

Overview
--------

Brings source code together to build and maintain programs

Examples
--------

- Looks for makefiles named GNUmakefile, makefile, or Makefile and updates all executables described in the makefile

    $make

- uses *MyMakefile* as a makefile

    $make -f MyMakefile
